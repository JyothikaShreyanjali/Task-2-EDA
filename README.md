# Titanic Dataset - Task 2 (EDA & Preprocessing)

## Objective
To perform Exploratory Data Analysis (EDA) and data preprocessing on the Titanic dataset.

## Steps Performed
- Loaded dataset
- Checked dataset information
- Handled missing values (Age, Embarked)
- Dropped Cabin column
- Encoded categorical variables (Sex, Embarked)
- Dropped unnecessary columns
- Created cleaned dataset

## Files in this project
- titanic-Dataset(2).csv (original dataset)
- cleaned_titanic.csv (processed dataset)
- Task2_EDA.ipynb (main notebook)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
